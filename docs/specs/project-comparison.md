# Feature Spec — Project Comparison

## Purpose

Project Comparison allows clients to evaluate multiple projects side by side using structured investment metrics.

This feature supports decision clarity.

---

## Access

Accessible from:

- Decision Workspace
- Project detail page

---

## Comparison Limit

Maximum projects per comparison:

3

---

## Comparison Attributes

The comparison table must include:

- Project Name
- Fit Score
- Investment Grade
- Price Range
- Rental Yield
- Risk Level
- Growth Potential

---

## Data Source

Comparison data is derived from:

- project intelligence tables
- matching results
- investment grading engine

---

## UX Design Principles

Comparison must emphasize:

- clarity
- investment relevance
- decision support

Avoid overwhelming the user with excessive data.

---

## Interaction

Users may:

- add projects
- remove projects
- reorder projects

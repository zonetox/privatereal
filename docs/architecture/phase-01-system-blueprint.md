# Phase 01 — System Blueprint
## Real Estate Decision Intelligence Platform

### 1. System Overview
The Real Estate Decision Intelligence Platform is a system designed to support client real estate decision-making through:
- Client profile construction.
- Project analysis.
- Project-to-profile matching.
- Decision support.
- Lifecycle tracking.

**Note:** This is an advisory platform, not a marketplace. It does not handle transactions or payments.

### 2. System Philosophy
- **Profile-first**: Mandatory demand profile before project discovery.
- **Insight-driven**: Focused on analysis and insights over raw data.
- **Decision-oriented**: Every feature serves the goal of supporting a decision.
- **Advisor-assisted**: Technology enhances but does not replace human advisory.

### 3. System Scope
**Included:**
- Client profile management.
- Project intelligence data.
- Matching & fit analysis.
- Decision workspace/tools.
- Purchase lifecycle tracking.

**Excluded:**
- Real estate transactions/brokerage.
- Online payments.
- Marketplace/Auctions.

### 4. User Roles
- **Client**: Builds profiles, explores fit, analyzes projects, tracks journey.
- **Advisor**: Reviews client profiles, recommends assets, supports decisions, monitors progress.
- **Admin**: Manages project data, market metrics, and system integrity.

### 5. System Architecture Modules
- **5.1 Client Intelligence**: Financial profiles, investment goals, risk appetite, lifestyle preferences. Output: *Client Investment Profile*.
- **5.2 Project Intelligence**: Standardized data (Infrastructure, Legal, Product, Market, Risks, Expert Analysis).
- **5.3 Matching Engine**: Calculates *Project Fit Score* based on profile-to-asset alignment.
- **5.4 Decision Workspace**: Comparative analysis, financial modeling, advisor notes, decision checklists.
- **5.5 Lifecycle Management**: Tracking milestones (Exploration, Site Visit, Reservation, Deposit, SPA, Payment Milestones).
- **5.6 Advisor Intelligence**: Client dashboards, recommendation tools, consulting logs.

### 6. Client Decision Journey
1. Account Creation.
2. Demand Profile Construction.
3. System Recommendation/Discovery.
4. Professional Analysis & Advisory consultation.
5. Decision Making.
6. Post-decision Lifecycle Tracking.

### 7. Key Outputs
- **Client Investment Profile**: Strategic needs and goals.
- **Project Fit Score**: Alignment metric.
- **Decision Workspace**: Analytical environment.

### 8. System Boundaries
The system strictly **does not** perform:
- Property transactions.
- Payment processing.
- E-signing of legal contracts.
